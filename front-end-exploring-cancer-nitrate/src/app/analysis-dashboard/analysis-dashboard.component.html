Introduction

The question we want to explore is: Is there a relationship between Nitrate in drinking water and Cancer?

  • Independent (Explanatory) Variable: Nitrate Concentration from water well samples (parts per million)
  • Dependent Variable: Cancer Rate from census tracts (percentage of population)
Methodology
This flow chart shows the process for our analysis. We start with two inputs: cancer rates from census tracts, and nitrate concentrations from well water samples (both from 2016). In order to explore the relationship between these factors they must be in the same form. To get the data in the same form we interpolate the well samples and use zonal statistics to get the average nitrate concentration per census tract. Once both cancer rates and average nitrate concentration are averaged per census tract we can run a linear regression. The "R-squared" value of the regression results will tell us how much of the cancer rate could be explained by nitrate concentration in drinking water.


The green-colored steps (Interpolation, Residuals, and Results) are the steps we can interact with in this app. The rest is done automatically for you. You can set the Interpolation parameters, run the full analysis, and explore the results.

Map of Regression Residuals
Using linear regression we can guess what the cancer rate will be in any given census tract based only on the nitrate concentration in water there. This map displays the difference (i.e. residual) between our guesses and the actual cancer rates in each tract.

Summary of Regression Results
The regression summary tells us 1) What the R-Squared value is, and 2) if we can trust the R-Squared value. An R-Squared value of 0.20, for example, would suggest that nitrate in drinking water could explain 20% of the cancer rate. The other statistics tell us if the R-Squared value can be trusted.
